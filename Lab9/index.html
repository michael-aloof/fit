Завдання A–D

Введення даних, обчислення результатів, повідомлення у вікні та на сторінці.

  1. AУніверситет

    Складіть програму, яка запитує у користувача за допомогою prompt назву ВНЗ, в якій він навчається, а потім після введення даних друкує на сторінці повідомлення.

    Приклад: “Я вчуся в ...”

  2. BПробіжка

    Дано розміри (в метрах) футбольного поля, що має форму прямокутника. Яку відстань пробіжить спортсмен, що “намотав” кругів навколо цього поля? Число n задається з клавіатури.

    Відстань = n × периметр поля = n × 2 × (a + b)

  3. CЗадом наперед

    Дано тризначне натуральне число. Вивести на екран число, що виходить при прочитанні вихідного числа праворуч наліво.

  4. DФормула

    Наведено деякі дійсні числа a і b. Обчислити значення виразу з точністю до сотих. Перевірка даних на коректність потрібна.

    z = sin(a² + b²) · (a / (a² + 25) + b) / (√b + (a + b) / 2)

    Для коректного обчислення: b ≥ 0 і знаменник не повинен дорівнювати нулю.
